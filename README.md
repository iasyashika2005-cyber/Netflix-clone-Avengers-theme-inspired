# Netflix-clone-Avengers-theme-inspired
A responsive Netflix-style landing page with an Avengers theme, built to 
practice CSS Flexbox, CSS Grid, and modern responsive design techniques.

## Features
- Dark-themed landing page inspired by OTT streaming platforms
- Responsive navigation bar built with Flexbox
- Hero/banner section with gradient overlay
- Movie/show poster grid built with CSS Grid, with hover effects
- Fully responsive across mobile, tablet, and desktop

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
